EasyConnect

A tabbed remote desktop application with a Chrome-like UI and support for RDP, VNC, SSH, and remote PowerShell connections.

Download Release Installer Download CI Installer Fork me on GitHub


Current version: 2.1.0

Efficient

Manage all of your remote connections from within a single tabbed interface with support for tab tearing to new windows and tab reorganization.

Extensible

You can extend EasyConnect by writing plugins for other remote desktop protocol. Plugins for RDP, VNC, SSH, and PowerShell are included out of the box.

Secure

Your saved passwords are encrypted with an RSA key stored in your machine's key store.

Microsoft Store

Available to install through the Microsoft Store.

Automatic updates

Automatic updating with the Microsoft Store or NetSparkle keeps you running on the latest version.

Open source

EasyConnect is open source software and is completely free for commercial use. It is licensed under the GPL.
